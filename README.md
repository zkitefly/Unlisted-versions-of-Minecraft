## 安装

### HMCL

将下载的文件直接拖动至 HMCl 启动页面

然后就弹出了安装

此时会弹出一个弹窗，请点击“是”

然后点击安装即可

![](/HMCL.gif)

### 其他启动器

打开启动器的 .minecraft 文件夹

将下载的文件解压，打开并找到 .minecraft 文件夹

将该 .minecraft 覆盖到你启动器的 .minecraft 文件夹即可
